# 🔍 竞品价格监控系统

自动采集电商平台商品价格，智能分析趋势，实时告警通知。

![蟒蛇](https://img.shields.io/badge/Python-3.10+-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-0.109-green)
![流式处理](https://img.shields.io/badge/Streamlit-1.31-red)

---

## ✨ 功能特性

-📦**多平台支持** - 淘宝、京东、拼多多商品监控
-📈**价格追踪** - 自动采集，历史记录，趋势图表
-🔔**智能告警** - 降价/涨价/阈值告警，邮件/微信通知
- ⚙️ **定时任务** - 自动调度，无需人工干预
-📊**数据看板**-Streamlit可视化界面

---

## 🛠️ 技术栈

| 类别 | 技术 |
|------|------|
| 后端框架 |FastAPI|
| 前端看板 |流式处理|
| 数据库 |SQLite / PostgreSQL|
| 爬虫 |要求+靓汤|
| 数据分析 |熊猫|
| 可视化 |Plotly|
| 定时任务 |APScheduler（高级计划调度器）|
